SG Bingo — Social Bingo Game

CategoryPersonal Project
Tech StackReact, TypeScript, Vite, Tailwind CSS
RoleSolo Developer
Livesgbingo.app

Project description

Mobile-first social bingo game built for friend groups and teams. Players generate unique bingo cards from event pools, spot real-world items or situations, and race to complete their board first. Designed for use at social events, office activities, and travel.

Key features

  • Unique card generation with grid index mapping
  • Multiple event pools (SG Life, Food, Office, Travel)
  • Photo proof system for event verification
  • Shared leaderboards and group scoring
  • Perpetual and weekly pool types
  • Koh Samui travel bingo integration
  • Auth with email and social login
  • Bingo blog with automated daily posts

Architecture

React SPA with Vite, Tailwind CSS for styling. Supabase for auth and database. Deployed on Vercel. Automated blog generation via OpenClaw cron jobs writing to a GitHub Pages blog with RSS feed.
