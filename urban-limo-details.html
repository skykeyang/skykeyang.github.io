Urban Limo — Luxury Transportation Marketplace

CategoryFull-Stack Project
Tech StackReact, TypeScript, FastAPI, PostgreSQL, AWS
RoleSolo Developer
Livehttps://urban-limo.com

Project description

Full-stack marketplace connecting customers with professional drivers for scheduled luxury transportation. Pull-based marketplace model.

Key features

  • Complete ride lifecycle
  • Dynamic pricing with rate cards and surcharges
  • Stripe payments with commission splits
  • Role-based portals
  • AWS ECS deployment
  • OpenAPI-first dev
  • 53+ backend tests

Architecture

Monorepo with Turborepo + pnpm. FastAPI + SQLAlchemy, React/TypeScript + Tailwind, Celery workers, Redis caching.
