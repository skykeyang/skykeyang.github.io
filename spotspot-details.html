Project description

Mobile-first social bingo app where groups compete on event-themed boards. Players record proof — photos and short video clips (up to 5s) — for each bingo square, track their progress, and celebrate bingo lines together with animations. Designed for friend groups at social events, trips, and team activities.

Group-based model with invite codes. Each group gets its own board within shared event pools. Pool switching requires 70% member vote. Real-time leaderboards track group and individual progress.

Key features

  • Group-based bingo boards with invite codes
  • Photo and video proof uploads (5s clips for future compilation)
  • Configurable media mode per group (photos only, videos only, or both)
  • Bingo line detection with celebration animations
  • Pool switching with 70% member vote threshold
  • Shared and individual real-time leaderboards
  • Auth with email and social login
  • Event discovery feed

Architecture

React SPA with Vite and Tailwind CSS. Supabase for auth and database with PostgreSQL and row-level security (RLS). Browser-side video compression using the MediaRecorder API. Deployed on Vercel.
